Press power button(Don’t release)

$ adb shell sendevent /dev/input/event2 1 $((0x74)) 1 $ adb shell sendevent /dev/input/event2 0 0 0

Release power button

$ adb shell sendevent /dev/input/event5 1 $((0x74)) 0 $ adb shell sendevent /dev/input/event5 0 0 0
