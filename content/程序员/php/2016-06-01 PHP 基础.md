# Php 设置时区
1. 使用date_default_timezone_set()设置 date_default_timezone_set('PRC');   date_default_timezone_set('Asia/shanghai');
2. ini_set('date.timezone','Asia/Shanghai');
3. 修改错误级别配置（不推荐）
4. 修改php.ini，显式设置date.timezone=PRC
